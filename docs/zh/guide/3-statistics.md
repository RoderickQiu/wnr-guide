# 时间统计功能

时间统计功能是wnr在1.17中加入的一项功能，让你可以清楚地看见每天你使用了wnr多久，把多少时间用在工作上，多少时间留给了休息，等等。

## 如何进入

和设置功能一样，Windows系统中点击左上角wnr图标，在下拉菜单中找到“使用统计”，macOS系统中则是在顶栏上找到操作->使用统计进入统计页面。
